📸 Responsive Image Gallery 
My Frontend Internship Project
Hey there! This is a responsive image gallery I built during my frontend internship. It's got all the cool stuff – lightbox view, next/prev buttons, hover effects, and even category filters (that was the bonus task!).<br><br>

🎯 What I Had to Do
Build an image gallery using HTML & CSS layout <br>
Add JavaScript for navigation and lightbox<br>
Make it look smooth with hover effects & transitions<br>
Make it work on phones, tablets, and desktops<br>
Bonus: image filters/categories<br><br>


✨ What It Does<br>
Filter images by Nature, City, Abstract, or People<br>
Click any image to open it in a fullscreen lightbox<br>
Next/Prev buttons to browse through pages of images<br>
Hover over images – they zoom in slightly and show the category tag<br>
Works on any screen size – I tested it on my phone and laptop<br><br>


🛠️ Tech Stack<br>
HTML5<br>
CSS3 (Grid, Flexbox, transitions)<br>
JavaScript (ES6)<br><br>


🚀 How to Run It<br>
Super simple:<br>
Download the image.html file<br>
Double-click it – it'll open in your browser<br>
That's it! No server needed, no dependencies to install<br>
Or if you're fancy, use Live Server in VS Code.<br>


🎮 How to Use<br>
What you want to do	How to do it<br>
Filter images	Click the category buttons on top<br>
Go to next page	Click "Next" button<br>
See image fullscreen	Click any image<br>
Navigate in lightbox	Click arrows or use ← → keys<br>
Close lightbox	Click ✕ or press Escape<br><br>


📱 Responsive Stuff<br>
On desktop: 3-4 columns of images<br>

On tablet: 2-3 columns<br>

On phone: 1-2 columns<br>

I used CSS Grid with auto-fill so it just... works.<br><br>


🧠 What I Learned<br>
This was my first real internship project and honestly I learned a ton:<br>
CSS Grid is a lifesaver for layouts<br>
Event delegation in JS (took me a while to figure out)<br>
Lightbox modals are trickier than they look<br>
Keyboard accessibility matters (arrow keys in lightbox)<br>
Always test on your phone before saying "it's responsive"<br>
