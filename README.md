🖼️ Project 1: Lumina Gallery
What it does
Image gallery with category filters, pagination, and a fullscreen lightbox. Click any image to open it big, use arrows to navigate.

Features
✅ Filter by: Nature, City, Abstract, People

✅ Pagination (6 images per page)

✅ Lightbox with Prev/Next + keyboard arrows

✅ Hover zoom + category badge

✅ Works on phone/tablet/desktop

Tech
HTML · CSS Grid · Vanilla JS<br><br>



🎵 Project 2: Harmony Flow Music Player<br>
What it does<br>
A sleek music player with playlist, progress bar, volume control, and autoplay. Pick a song, play it, skip tracks, drag the progress bar.

Features<br>
✅ Play/Pause, Next/Prev buttons

✅ Clickable progress bar (seek)

✅ Volume slider

✅ Playlist with 4 sample songs

✅ Autoplay toggle (ON/OFF)

✅ Current time & duration display

✅ Auto-advance to next song

Tech
HTML · CSS · JavaScript (Audio API)<br><br>

🧮 Project 3: Precision Calculator<br>
What it does<br>
A modern calculator that handles basic math. Works with mouse clicks AND keyboard.<br>

Features
✅ Addition, subtraction, multiplication, division

✅ Percentage and modulo (%)

✅ Clear (AC) and Delete (⌫) buttons

✅ Decimal support

✅ Keyboard support (numbers, operators, Enter, Escape, Backspace)

✅ Division by zero error handling

Tech
HTML · CSS Grid · JavaScript


🎮 Quick Demo Guide
Image Gallery
text
Click filter buttons → See only Nature/City/Abstract/People
Click any image → Lightbox opens
Use ◀ ▶ in lightbox → Browse all filtered images
Press ESC → Close lightbox
Use Prev/Next buttons below gallery → Change page
Music Player
text
Click Play → Song starts
Drag progress bar → Jump to any timestamp
Adjust volume → Right side slider
Click any song in playlist → Switch tracks
Toggle Autoplay → Player loops or stops after last song
Calculator
text
Click buttons or use keyboard:
0-9, ., +, -, *, /, % 
Enter or = → Calculate
Escape → Clear all (AC)
Backspace → Delete last character
